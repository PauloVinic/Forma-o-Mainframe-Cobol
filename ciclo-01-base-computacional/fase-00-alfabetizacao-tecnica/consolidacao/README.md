# Consolidação da Fase 0 — Alfabetização Técnica

Esta pasta reúne os materiais de consolidação da Fase 0. A consolidação não substitui as 12 aulas principais; ela organiza revisão, fixação, verificação de domínio e preparação para uma decisão responsável sobre avanço futuro para a Fase 1 — Pensamento Computacional.

A Fase 0 construiu vocabulário básico sobre tecnologia da informação, sistemas, hardware, software, pessoas, dados, processamento, entrada, saída, programas, código-fonte, execução, arquivos, ambientes e erros. A consolidação existe para verificar se esses conceitos foram compreendidos como relações, e não apenas memorizados como definições isoladas.

## Ordem recomendada de uso

1. Leia `resumo-geral-da-fase.md` para retomar a progressão das 12 aulas.
2. Use `guia-de-revisao.md` para revisar por blocos conceituais.
3. Estude `mapa-integrado-de-conceitos.md` para enxergar relações entre os temas.
4. Trabalhe o `estudo-dirigido.md` com respostas próprias, sem copiar frases das aulas.
5. Resolva `exercicios-teoricos.md` e registre dúvidas.
6. Responda `perguntas-dissertativas.md` para testar explicação desenvolvida.
7. Faça `avaliacao-objetiva.md` e corrija pelo gabarito comentado.
8. Faça `avaliacao-dissertativa.md` usando a `rubrica-de-correcao.md`.
9. Preencha `checklist-final-de-dominio.md`.
10. Use `plano-de-estudo-sugerido.md` para organizar revisão, retomada de lacunas e eventual reavaliação.

O resultado esperado não é avançar automaticamente. O aluno deve usar esses materiais para descobrir se consegue explicar, comparar e aplicar os conceitos da Fase 0 em situações simples. Se houver lacunas importantes, a próxima ação deve ser revisar aulas e refazer partes da consolidação antes de iniciar a Fase 1.
