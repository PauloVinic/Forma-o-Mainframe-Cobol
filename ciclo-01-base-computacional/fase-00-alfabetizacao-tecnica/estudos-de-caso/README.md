# Estudos de Caso da Fase 0

Esta pasta receberá estudos de caso simples da Fase 0 futuramente. Eles deverão usar situações de empresas, bancos, governo ou sistemas administrativos para mostrar por que os conceitos básicos importam.

Os estudos de caso poderão abordar situações como cadastro de clientes, emissão de relatório, atualização de benefício, consulta de saldo, controle de estoque ou falha em rotina administrativa. O objetivo será fazer o aluno reconhecer dados, entradas, processamento, saídas, arquivos, ambientes e erros em contextos plausíveis.

Não devem ser criados estudos de caso completos nesta etapa. Quando forem desenvolvidos, eles precisarão permanecer no nível conceitual da Fase 0, sem antecipar COBOL, JCL, DB2, CICS ou práticas de desenvolvimento.
