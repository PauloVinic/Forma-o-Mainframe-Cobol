# Exercícios da Fase 0

Esta pasta receberá exercícios teóricos da Fase 0 futuramente. Os exercícios deverão testar compreensão conceitual, comparação entre conceitos e uso correto do vocabulário técnico.

Como a Fase 0 antecede programação, os exercícios não devem envolver código. Eles poderão pedir que o aluno diferencie termos, classifique situações, identifique entradas e saídas, reconheça tipos de erro, explique o papel de arquivos ou descreva um sistema simples em linguagem técnica inicial.

Os exercícios completos ainda não devem ser criados nesta etapa. Quando forem criados, precisarão estar alinhados ao roteiro das aulas, aos critérios de domínio da fase e às diretrizes editoriais do projeto.
