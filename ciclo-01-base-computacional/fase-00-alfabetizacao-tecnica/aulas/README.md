# Aulas da Fase 0

Esta pasta reúne as aulas completas da Fase 0 — Alfabetização Técnica. As aulas deverão ser criadas progressivamente, uma por prompt ou em pequenos grupos, sempre respeitando o planejamento da fase.

Cada aula futura deverá seguir `templates/TEMPLATE_AULA.md` e respeitar `DIRETRIZES_EDITORIAIS.md`. Isso significa que a aula não poderá ser apenas uma lista de tópicos, um resumo curto ou uma definição telegráfica. Cada conceito importante deverá ser explicado com desenvolvimento, exemplos conceituais, transições e conexão com a progressão da formação.

## Conjunto de aulas concluído

As 12 aulas planejadas da Fase 0 — Alfabetização Técnica foram desenvolvidas. A próxima etapa não deve iniciar a Fase 1 ainda; deve consolidar a Fase 0 com revisão geral, estudo dirigido, exercícios teóricos, avaliação e checklist final.

Nenhuma aula desta fase deve introduzir código, COBOL, comandos de ferramenta ou ambiente de desenvolvimento prático antes do momento previsto na formação.
