# Prompts do Projeto

Esta pasta guardará os prompts usados na construção da formação. O objetivo é manter histórico das decisões, preservar a intenção pedagógica de cada etapa e facilitar revisões futuras.

Cada prompt registrado deve indicar o que foi pedido, quais limites foram definidos e que parte do projeto ele ajudou a construir. Isso evita perda de contexto e reduz o risco de prompts futuros contradizerem as diretrizes do projeto.

Os prompts desta pasta não substituem os documentos de orientação. Antes de usar ou criar um novo prompt, consulte também README.md, ROADMAP.md, DIRETRIZES_DO_PROJETO.md, DIRETRIZES_EDITORIAIS.md e os documentos em planejamento/.
