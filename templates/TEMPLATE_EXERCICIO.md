# Template de Exercicio

Use este template somente quando a fase já permitir exercícios. Nas fases iniciais, exercícios podem ser conceituais, de leitura, classificação, explicação ou análise de situação. Não criar exercícios de código antes da hora.

## Objetivo do exercicio

Explique que compreensão o exercício pretende verificar ou desenvolver. O objetivo deve estar ligado à fase e aos critérios de domínio.

## Enunciado

Apresente o problema com contexto suficiente. O enunciado deve ser claro, completo e alinhado ao conteúdo já estudado.

## Dados ou situacao fornecida

Descreva os dados, textos, cenários ou condições que o aluno deve analisar. Não introduza elementos que dependam de fases futuras sem explicação.

## Tarefa do aluno

Explique o que deve ser entregue. A tarefa pode pedir explicação, classificação, comparação, desenho conceitual, identificação de erro ou justificativa.

## Criterios de avaliacao

Indique como a resposta será avaliada. Os critérios devem considerar clareza, precisão, uso correto do vocabulário, conexão com o conteúdo e justificativa.

## Comentarios esperados para correcao

Registre pontos que uma boa resposta deve contemplar. Esta seção não deve virar gabarito superficial; ela deve orientar revisão e feedback.
