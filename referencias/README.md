# Referencias

Esta pasta reunirá referências oficiais, bibliografia, documentação técnica e materiais complementares ao longo da construção da formação.

As referências devem ser adicionadas progressivamente, conforme cada fase exigir. A preferência deve ser por fontes oficiais, livros sólidos, documentação técnica reconhecida e materiais que ajudem a aprofundar conceitos sem quebrar a progressão pedagógica.

Referências não devem ser usadas apenas para acumular links. Cada fonte incluída futuramente deve ter relação clara com uma fase, aula, estudo dirigido ou decisão técnica do projeto.

Quando forem adicionadas referências sobre Mainframe, COBOL, z/OS, JCL, VSAM, DB2 ou CICS, elas devem ser contextualizadas para que o aluno entenda em que momento da formação aquele material se torna útil.
